ECL

 ECL (params, lr=<required parameter>, momentum=0, dampening=0,
      weight_decay=0, nesterov=False, maximize=False,
      foreach:Optional[bool]=None, clip_angle_value=0)

Base class for all optimizers.

.. warning:: Parameters need to be specified as collections that have a deterministic ordering that is consistent between runs. Examples of objects that don’t satisfy those properties are sets and iterators over values of dictionaries.

Args: params (iterable): an iterable of :class:torch.Tensor s or :class:dict s. Specifies what Tensors should be optimized. defaults: (dict): a dict containing default values of optimization options (used when a parameter group doesn’t specify them).
